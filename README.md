**Patient Data Dashboard**
<p>In this project I used synthetic Massachusetts clinical data to create a healthcare dashboard. Some key information in the spread includes KPI's (Inpatient ALOS, ER turnaround time), a patient demographic histogram, time-series analysis of hospital encounters, and a visualization of encounters per payer.<p>
